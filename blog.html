Stack MERN - Blog application

Fully functional Blog with MongoDB and NodeJS

The Project

This application is a classic blog, and so it's a really good example of a training fullstack app. It offers basic functionalities: displays posts list (with pagination), displays single post, adding posts. This was strictly guided by psd files provided.

Task was sort of a scenario based, meaning that there was imaginary client that had a vision of what the project needs to look like and all his wishes have to be fulfilled. Whole thing was a simulation of how would the project be handled by a developer company. It was fun!
